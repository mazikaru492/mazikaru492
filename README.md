### About Me

京都デザイン＆テクノロジー専門学校で勉強しながら、Web開発とセキュリティを軸に色々つくっています。
きれいなUIとセキュアなコード、両方にこだわりたいタイプです。

最近は **Next.js** と **TypeScript** をメインで触っていて、
空き時間に WiFi 解析ツールやセキュリティ関連の小ツールをつくるのが趣味になってます。
